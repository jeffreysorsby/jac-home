Proceso para carga de archivos

1. Ingresar al apartado S3 File Admin y seleccionar la carpeta iconos

2. Seleccionar Subir archivo

3. Seleccionar archivo y Salvar

4. Una vez creado, ir al apartado de Document y seleccionar Crear

5. Llenar los campos correspondientes:

    • Nombre (el nombre que se mostrara en la página)
    • Url (siendo la dirección que se desea abrir), siempre formada por: “https://s3.amazonaws.com/jachome.mx/” + "ruta/del-archivo.jpg"
    • Image Url (siendo la imagen que aparecerá en la página), siempre formada por “https://s3.amazonaws.com/jachome.mx/” + "ruta/del-archivo.jpg"
    • Doc Type (Formato del documento)
    • Car (el modelo al cual será asignado)
    • Salvar

6. Verificar
